# Automation Features

This directory contains 19 feature(s) in the Automation category.

## Features

- [Auto Scene Break Detection](./auto-scene-break-detection/overview.md) - Automatically detect scene changes in chat messages.
- [Auto Scene Break on Load](./auto-scene-break-on-load/overview.md) - Run detection when chat is loaded.
- [Auto Scene Break on New Message](./auto-scene-break-on-new-message/overview.md) - Run detection when new messages arrive.
- [Auto Scene Break Generate Recap](./auto-scene-break-generate-recap/overview.md) - Automatically generate recap when scene detected.
- [Scene Detection Prompt Customization](./scene-detection-prompt-customization/overview.md) - Customize prompt used to detect scene breaks.
- [Scene Detection Message Offset](./scene-detection-message-offset/overview.md) - Skip latest N messages when detecting (avoid false positives).
- [Minimum Scene Length Enforcement](./minimum-scene-length-enforcement/overview.md) - Require minimum messages between scenes (default: 3).
- [Scene Detection Validation](./scene-detection-validation/overview.md) - Advanced logic to validate time/location/cast transitions.
- [Rationale Format Validation](./rationale-format-validation/overview.md) - Prevent false positives from decorative separators.
- [Auto-Generate Scene Names (Auto-Detection)](./auto-generate-scene-names-auto-detection/overview.md) - Generate names when auto-detecting scenes.
- [Auto-Generate Scene Names (Manual)](./auto-generate-scene-names-manual/overview.md) - Generate names when manually creating scenes.
- [Auto-Hide Messages](./auto-hide-messages/overview.md) - Automatically hide messages from scenes older than X scenes.
- [Scene Detection Rationale Format Validation](./scene-detection-rationale-format-validation/overview.md) - Validate rationale format to prevent false positives from decorative separators.
- [Scene Detection Objective Shift Detection](./scene-detection-objective-shift-detection/overview.md) - Detect significant narrative objective shifts indicating scene changes.
- [Scene Detection Continuity Veto System](./scene-detection-continuity-veto-system/overview.md) - Advanced veto logic for scene continuity and objective validation.
- [Clear Checked Flags in Range](./clear-checked-flags-in-range/overview.md) - Clear auto scene break checked flags for specific message range.
- [Set Checked Flags in Range](./set-checked-flags-in-range/overview.md) - Set auto scene break checked flags for specific message range.
- [Clear All Checked Flags](./clear-all-checked-flags/overview.md) - Clear all auto scene break checked flags across entire chat.
- [Manual Scene Break Detection Command](./manual-scene-break-detection-command/overview.md) - Manually trigger scene detection for specific message ranges.
