# Scene Management Features

This directory contains 15 feature(s) in the Scene Management category.

## Features

- [Scene Break Markers](./scene-break-markers/overview.md) - Persistent scene break data on messages.
- [Scene Break Visibility Toggle](./scene-break-visibility-toggle/overview.md) - Show/hide scene break markers.
- [Scene Break Hash Tracking](./scene-break-hash-tracking/overview.md) - Detect when recap content changes.
- [Scene Recap Metadata](./scene-recap-metadata/overview.md) - Store metadata about scene recaps.
- [Scene Recap Current Index](./scene-recap-current-index/overview.md) - Track active version of scene recap.
- [Scene Recap Include/Exclude](./scene-recap-include-exclude/overview.md) - Control which scene recaps are injected.
- [Scene Message Type Filtering](./scene-message-type-filtering/overview.md) - Choose user/AI/both messages for scene recaps.
- [Scene Message History Mode](./scene-message-history-mode/overview.md) - Configure which messages included in scene recap context.
- [Scene Message History Count](./scene-message-history-count/overview.md) - Set how many messages to include.
- [Auto Scene Break Checked Flags](./auto-scene-break-checked-flags/overview.md) - Track which messages have been checked for scenes.
- [Scene Break Rescan Capability](./scene-break-rescan-capability/overview.md) - Force full rescan of all messages for scenes.
- [Scene Navigator Jump-to-Message](./scene-navigator-jump-to-message/overview.md) - Click scene in navigator to jump to it in chat.
- [Scene Break Hash Verification](./scene-break-hash-verification/overview.md) - Verify scene recap content hasn't changed using hash comparison.
- [Scene Metadata Tracking](./scene-metadata-tracking/overview.md) - Store and retrieve comprehensive scene metadata for each scene break.
- [Scene Recap Version Management](./scene-recap-version-management/overview.md) - Manage multiple versions of scene recaps with current version tracking.
