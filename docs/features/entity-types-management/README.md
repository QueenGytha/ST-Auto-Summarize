# Entity Types Management Features

This directory contains 10 feature(s) in the Entity Types Management category.

## Features

- [Entity Type Configuration](./entity-type-configuration/overview.md) - Configure which entity types to extract from recaps.
- [Entity Type Parsing](./entity-type-parsing/overview.md) - Parse entity types from extraction results with validation.
- [Entity Type Normalization](./entity-type-normalization/overview.md) - Normalize entity type names for consistency.
- [Entity Type Sanitization](./entity-type-sanitization/overview.md) - Sanitize entity type values to prevent invalid entries.
- [Entity Type Definition Parsing](./entity-type-definition-parsing/overview.md) - Parse entity type definitions from raw configuration strings.
- [Entity Type Map Creation](./entity-type-map-creation/overview.md) - Create entity type mapping structures for processing.
- [Entity Type Flags Application](./entity-type-flags-application/overview.md) - Apply entity-specific flags to lorebook entries.
- [Entity Type Name Sanitization](./entity-type-name-sanitization/overview.md) - Sanitize entity type names for use in lorebook keys.
- [Entity Type Restore Defaults UI](./entity-type-restore-defaults-ui/overview.md) - UI button to restore entity types to default configuration.
- [Entity Type Add/Remove UI](./entity-type-add-remove-ui/overview.md) - UI controls to add and remove custom entity types.
