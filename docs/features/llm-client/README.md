# LLM Client Features

This directory contains 4 feature(s) in the LLM Client category.

## Features

- [Unified LLM Client](./unified-llm-client/overview.md) - Centralized client for all LLM API requests.
- [LLM Call Parameter Validation](./llm-call-parameter-validation/overview.md) - Validate all LLM call parameters before sending requests.
- [Profile Resolution](./profile-resolution/overview.md) - Resolve connection profile IDs to actual profile configurations.
- [Preset Validity Checking](./preset-validity-checking/overview.md) - Verify completion preset exists before making LLM calls.
