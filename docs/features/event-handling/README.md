# Event Handling Features

This directory contains 16 feature(s) in the Event Handling category.

## Features

- [Chat Changed Event](./chat-changed-event/overview.md) - Load profile, refresh memory, scene detection on chat change.
- [Chat Deleted Event](./chat-deleted-event/overview.md) - Delete auto-created lorebook when chat deleted.
- [Group Chat Deleted Event](./group-chat-deleted-event/overview.md) - Handle group chat deletions.
- [Message Deleted Event](./message-deleted-event/overview.md) - Clean up running recaps on message delete.
- [Before Message Event](./before-message-event/overview.md) - Track before message generation.
- [User Message Event](./user-message-event/overview.md) - Track user messages.
- [Character Message Event](./character-message-event/overview.md) - Trigger scene detection on character messages.
- [Message Swiped Event](./message-swiped-event/overview.md) - Handle swipe logic and memory clearing.
- [Message Edited Event](./message-edited-event/overview.md) - Track message edits.
- [Message Received Event](./message-received-event/overview.md) - Track message received reasons (swipe vs normal).
- [More Messages Loaded Event](./more-messages-loaded-event/overview.md) - Refresh UI when loading more messages.
- [Group Selected Event](./group-selected-event/overview.md) - Update character enable button states.
- [Group Updated Event](./group-updated-event/overview.md) - Update character enable button states.
- [Chat Completion Prompt Ready](./chat-completion-prompt-ready/overview.md) - Inject metadata into prompts.
- [World Info Activated Event](./world-info-activated-event/overview.md) - Track which lorebook entries activated.
- [Generation Started Event](./generation-started-event/overview.md) - Track generation type and target message.
