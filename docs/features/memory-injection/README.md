# Memory Injection Features

This directory contains 7 feature(s) in the Memory Injection category.

## Features

- [Memory Injection Flow](./memory-injection-flow/overview.md) - End-to-end process for injecting recap memories into LLM prompts during chat.
- [Running Scene Recap Injection](./running-scene-recap-injection/overview.md) - Injects running scene recap into LLM prompt at configurable position.
- [Injection Position Control](./injection-position-control/overview.md) - Choose where memory is injected (Before/After Character Defs, Author's Note, etc.).
- [Injection Depth Control](./injection-depth-control/overview.md) - Set how deep in context memory appears.
- [Injection Role Control](./injection-role-control/overview.md) - Set role for injected memory (System, User, Assistant).
- [World Info Scanning](./world-info-scanning/overview.md) - Make memories available for world info scans.
- [Injection Preview](./injection-preview/overview.md) - Preview exactly what text will be injected into prompts.
