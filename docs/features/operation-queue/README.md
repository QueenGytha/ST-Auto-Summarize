# Operation Queue Features

This directory contains 15 feature(s) in the Operation Queue category.

## Features

- [Persistent Operation Queue](./persistent-operation-queue/overview.md) - Durable queue stored in lorebook (survives page reload).
- [Queue Processor](./queue-processor/overview.md) - Sequential execution of queued operations.
- [Queue Status Tracking](./queue-status-tracking/overview.md) - Track pending/in-progress/completed/failed operations.
- [Queue Pause/Resume](./queue-pause-resume/overview.md) - Manually pause and resume queue processing.
- [Queue Clear All](./queue-clear-all/overview.md) - Remove all operations from queue.
- [Queue Retry Logic](./queue-retry-logic/overview.md) - Automatic retry for failed operations.
- [Queue Blocking Mode](./queue-blocking-mode/overview.md) - Block chat input while queue processes operations.
- [Queue Progress UI](./queue-progress-ui/overview.md) - Real-time progress display for queue operations.
- [Queue Version Control](./queue-version-control/overview.md) - Invalidate in-flight operations on clear.
- [Queue Operation Timeout](./queue-operation-timeout/overview.md) - Timeout protection for stuck operations.
- [Queue Polling Interval](./queue-polling-interval/overview.md) - Configurable polling for queue updates.
- [Chat Blocking Toggle](./chat-blocking-toggle/overview.md) - Automatically block/unblock chat based on queue state.
- [Enter Key Interception](./enter-key-interception/overview.md) - Block Enter key when queue is processing.
- [Send Button Hiding](./send-button-hiding/overview.md) - Hide SillyTavern's send button during queue operations.
- [Queue Indicator Button](./queue-indicator-button/overview.md) - Custom button showing queue is active.
