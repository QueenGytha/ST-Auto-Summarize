# Slash Command Features

This directory contains 14 feature(s) in the Slash Command category.

## Features

- [/get_memory_enabled](./get-memory-enabled/overview.md) - Check if extension enabled in current chat.
- [/toggle_memory](./toggle-memory/overview.md) - Toggle extension on/off for current chat.
- [/toggle_memory_popout](./toggle-memory-popout/overview.md) - Open/close popout settings window.
- [/toggle_memory_injection_preview](./toggle-memory-injection-preview/overview.md) - Preview memory injection.
- [/get_memory <n>](./get-memory-n/overview.md) - Get memory for specific message.
- [/auto_recap_log_chat](./auto-recap-log-chat/overview.md) - Log current chat to console.
- [/auto_recap_log_settings](./auto-recap-log-settings/overview.md) - Log current settings to console.
- [/hard_reset](./hard-reset/overview.md) - Reset all settings to defaults.
- [/log_scene_recap_injection](./log-scene-recap-injection/overview.md) - Log scene recap injection settings.
- [/queue-status](./queue-status/overview.md) - Show operation queue status.
- [/queue](./queue/overview.md) - Alias for queue-status.
- [/queue-pause](./queue-pause/overview.md) - Pause the operation queue.
- [/queue-resume](./queue-resume/overview.md) - Resume the operation queue.
- [/queue-clear-all](./queue-clear-all/overview.md) - Clear all operations from queue.
