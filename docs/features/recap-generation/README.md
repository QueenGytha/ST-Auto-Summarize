# Recap Generation Features

This directory contains 13 feature(s) in the Recap Generation category.

## Features

- [Scene Recap Generation](./scene-recap-generation/overview.md) - Generates AI-powered summaries for scene breaks marked by the user.
- [Scene Recap Versioning](./scene-recap-versioning/overview.md) - Stores multiple versions of scene recaps, allowing users to switch between them.
- [Scene Recap Validation](./scene-recap-validation/overview.md) - Optional second LLM pass to validate scene recap format and quality.
- [Scene Recap Retry Logic](./scene-recap-retry-logic/overview.md) - Automatically retries recap generation with validation if quality check fails.
- [Running Scene Recap](./running-scene-recap/overview.md) - Combines multiple scene recaps into single cohesive narrative memory (enabled by default).
- [Running Scene Recap Versioning](./running-scene-recap-versioning/overview.md) - Maintains version history for running scene recaps with edit capability.
- [Running Scene Recap Auto-Generation](./running-scene-recap-auto-generation/overview.md) - Automatically regenerates when new scene recaps are created.
- [Running Scene Recap Exclude Latest N](./running-scene-recap-exclude-latest-n/overview.md) - Waits N scenes before including in running recap (default: 1).
- [Custom Recap Prompts](./custom-recap-prompts/overview.md) - Fully customizable prompts for scene and running recaps with macro support.
- [Recap Prefill](./recap-prefill/overview.md) - Configurable prefill text to guide LLM output format.
- [Connection Profile Selection](./connection-profile-selection/overview.md) - Use different API connections for recaps vs main chat.
- [Completion Preset Selection](./completion-preset-selection/overview.md) - Use different completion presets for recaps with custom temperature/tokens.
- [Include Preset Prompts Toggle](./include-preset-prompts-toggle/overview.md) - Control whether preset prompts are included in recap generation.
