# UI/Visual Features

This directory contains 22 feature(s) in the UI/Visual category.

## Features

- [Scene Break Visual Markers](./scene-break-visual-markers/overview.md) - Visual scene break dividers below messages.
- [Scene Break Button](./scene-break-button/overview.md) - Message menu button to manually mark scene breaks.
- [Scene Break Name Input](./scene-break-name-input/overview.md) - Add custom names to scene breaks (like chapter titles).
- [Scene Break Recap Display](./scene-break-recap-display/overview.md) - Shows recap text directly below scene break marker.
- [Scene Break Collapse/Expand](./scene-break-collapse-expand/overview.md) - Collapsible scene break sections.
- [Scene Navigator Bar](./scene-navigator-bar/overview.md) - Floating sidebar showing all scenes with navigation.
- [Navigator Bar Width Customization](./navigator-bar-width-customization/overview.md) - Adjust navigator width (30-500 pixels).
- [Navigator Font Size Customization](./navigator-font-size-customization/overview.md) - Adjust font size for scene names.
- [Navigator Bar Toggle](./navigator-bar-toggle/overview.md) - Show/hide navigator bar.
- [Scene Name Auto-Generation](./scene-name-auto-generation/overview.md) - Automatically generate brief scene names from recaps.
- [Running Scene Recap Navbar Controls](./running-scene-recap-navbar-controls/overview.md) - Floating navbar with version selector, edit, and regenerate buttons.
- [Popout Settings Window](./popout-settings-window/overview.md) - Draggable floating window for extension configuration.
- [Settings Panel](./settings-panel/overview.md) - Comprehensive settings UI with all controls.
- [Memory Editor Interface](./memory-editor-interface/overview.md) - Dedicated UI for viewing/editing all memories in chat.
- [Progress Bar UI](./progress-bar-ui/overview.md) - Visual progress indicators during recap generation.
- [Queue Status UI](./queue-status-ui/overview.md) - Shows current operation queue status and progress.
- [Lorebook Viewer](./lorebook-viewer/overview.md) - View active and inactive lorebook entries per message via button.
- [Lorebook Entry Icons](./lorebook-entry-icons/overview.md) - Scene breaks show icon to view associated lorebook entries.
- [Toast Notifications](./toast-notifications/overview.md) - User-friendly notifications for operations and errors.
- [Toast Duration Calculation](./toast-duration-calculation/overview.md) - Automatically calculate toast display duration based on content length.
- [Scene Break Icon Creation](./scene-break-icon-creation/overview.md) - Create and display lorebook icons for scene breaks.
- [Extension Reload Test Marker](./extension-reload-test-marker/overview.md) - Test marker UI for verifying extension reload functionality.
